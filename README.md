# Sistema-de-cadastro-para-clinica
Projeto de Extensão Acadêmica - Sistema de Cadastro desenvolvido em Python/SQLite com auxílio da Inteligência artificial

## 🛠️ Tecnologias Utilizadas
- Python 
- Interface Gráfica: Tkinter
- Banco de Dados: SQLite

## 🤖 Uso de Inteligência Artificial
A Inteligência Artificial Generativa (ChatGPT) foi integrada ao fluxo de trabalho para:
1. Modelagem lógica do banco de dados relacional.
2. Criação e otimização de scripts de validação de dados (como validação de formato de CPF).
3. Engenharia de prompts para acelerar o desenvolvimento do design de interface em Tkinter.
